Function Documentation¶

string ucs4_to_utf8(string dest, runic src)¶

Convert UCS-4 to UTF-8, will try to NULL-terminate, if there is space. No error checking is done. dest.len >= 4 * src.len + 4 To ensure a fit. Ensure enough space in dest string.

Return

Slice of dest with converted UTF-8 bytes with correct length.

Parameters

  • [out] dest: Empty string structure.

  • [in] src: UCS-4/UTF-32 encoded string.
